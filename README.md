## Цели проекта

Разработка моделей машинного обучения для:
1. **Прогноза механических свойств** композитов:
   - Модуль упругости при растяжении, ГПа
   - Прочность при растяжении, МПа
2. **Рекомендации** технологического параметра:
   - Соотношение матрица-наполнитель


## Данные

Исходные данные:
1. [`X_bp.xlsx`](X_bp.xlsx) — параметры связующего и итоговые свойства
2. [`X_nup.xlsx`](X_nup.xlsx) — параметры армирующего компонента (нашивки)
<br>

- **Объединение**: `inner join` по индексу  
- **Размер данных**: 1023 строки × 13 признаков  
- **Пропуски и дубликаты**: отсутствуют  
- **Предобработка**:
  + Бинаризация «Угол нашивки» (0 -> 0, 90 -> 1)
  + Удаление выбросов по методу IQR
  + Нормализация через `StandardScaler`

## Модели и результаты

### 1. Прогноз механических свойств
- **10 алгоритмов** сравнено: от `LinearRegression` до `GradientBoosting`
- **Оптимизация**: `GridSearchCV` с `cv=10`
- **Метрики**: MAE, R²
- **Лучшие модели**:
  - **Модуль упругости**: `SVR` (MAE ~ 0.86)
  - **Прочность**: `ElasticNet` (MAE ~ 360.6)

> Визуализация: барчарты сравнения, важность признаков

### 2. Нейронная сеть для «Соотношение матрица-наполнитель»
- **Архитектура**: 6 полносвязных слоёв + BatchNorm + Dropout
- **Нормализация**: встроенный слой `Normalization`
- **Оптимизатор**: Adam
- **Результат**:
  - **MAE модели**: 0.7448
  - **MAE среднего**: 0.6812
  - Модель **превосходит** базовый прогноз

> Графики: потери, фактические vs предсказанные значения, scatter-plot

## Веб-приложение

### Функционал
- Ввод 12 параметров композита
- Получение прогноза **«Соотношение матрица-наполнитель»**
- Кнопки **«Готово»** и **«Очистить»**
- Очистка введенных значений при необходимости

### Технологии
- **Бэкенд**: Flask, TensorFlow
- **Фронтенд**: HTML/CSS 
- **Модель**: сохранена в формате `.keras`

### Как запустить

1. Установите зависимости:
   ```bash
   pip install flask tensorflow pandas numpy
2. Скачайте и распакуйте архив [`composite_app.zip`](composite_app.zip)
3. Перейдите в папку `app` и кликните на адресную строку:
   ```bash
   cmd
4. Запустите сервер:
   ```bash
   python app.py
5. Откройте сайт [`http://127.0.0.1:5000`](http://127.0.0.1:5000)
